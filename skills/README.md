# Repository-Local Skills

This directory is for repository-local helper skills used while developing or
operating this monorepo.

Current skills:

- `github-ci-triage`: inspect and diagnose GitHub Actions CI, release, deploy,
  and workflow failures.

Productized client-specific skill packages should live under their owning
package. The OpenClaw CALL-E CLI skill source lives in
`packages/openclaw-cli-skill/skills/`.
